# MiniMax

Prompts the user for 10 integers and prints the biggest and smallest numbers.

## Execution

```bash
Enter num #1: 2
Enter num #2: 23
Enter num #3: 1
Enter num #4: 5
Enter num #5: 78
Enter num #6: 2
Enter num #7: 6
Enter num #8: 35
Enter num #9: 2
Enter num #10: 1
Biggest number: 78
Smallest number: 1
```
